# Preview Assets

Preview images and GIFs are hosted as GitHub Release assets to keep the main skill checkout lightweight.

- Release: https://github.com/ChuluuMGL/business-website-skill/releases/tag/preview-assets-v1.3.0
- Style overview: https://github.com/ChuluuMGL/business-website-skill/releases/download/preview-assets-v1.3.0/style-overview.png
- AI concept moodboard: https://github.com/ChuluuMGL/business-website-skill/releases/download/preview-assets-v1.3.0/ai-style-moodboard.jpg

Agents do not need these binary previews to execute the skill. Use `../presets/`, `../../references/`, `../../scripts/`, and `../templates/` for the actual workflow.

To regenerate previews locally:

```bash
python3 scripts/generate_preview_assets.py
```
